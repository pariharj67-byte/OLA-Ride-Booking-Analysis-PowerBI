# 🚖 OLA Ride Booking Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes OLA ride booking data using Power BI to uncover insights related to booking trends, revenue, cancellations, vehicle performance, and customer satisfaction.

---

## 📊 Dashboard Features
- Total Bookings, Successful Bookings, Cancellation Rate
- Ride Volume Trend (Daily)
- Revenue Analysis by Payment Method
- Vehicle-wise Performance
- Cancellation Analysis (Customer & Driver)
- Driver & Customer Ratings

---

## 🧹 Data Cleaning
- Missing numeric values replaced with average
- Ratings handled using median
- Null categorical values replaced with "Other"
- Incomplete ride fields handled appropriately

---

## 📈 Key Insights
- High booking volume (100K+) indicates strong demand
- Cancellation rate (~28%) is a major concern
- Cash & UPI dominate payment methods
- Ratings are stable (~4.0) but can be improved

---

## 💡 Business Recommendations
- Reduce cancellations through better driver allocation
- Improve driver accountability
- Promote digital payments
- Enhance customer experience
